# Free Email Validator

**Free Email Validator** is a simple tool designed to verify and validate email addresses.
This tool focuses on **business/domain emails only** and does **not allow free email providers** such as Gmail, Yahoo, or Outlook.

The system helps users clean and verify email lists to improve marketing performance and email deliverability.

## Features

* Validate business email addresses
* Block free email providers (Gmail, Yahoo, Hotmail, etc.)
* Fast validation process
* Simple web interface
* Lightweight PHP-based system

1. Download or clone the repository.

# How to Use

1. Run **index.php** in your browser.
2. Import or paste your email list.
3. The system will automatically:

   * Validate email format
   * Check domain emails
   * Block free email providers
4. View or export the validated results.

## Important Note

This validator accepts **only domain-based emails**, such as:

```
name@company.com
info@business.org
contact@domain.net
```

Free email providers are **not allowed**, including:

```
gmail.com
yahoo.com
hotmail.com
outlook.com
```
